# 🤖 Telegram-бот для Битрикс24

Бот позволяет прямо из Telegram:
- 📋 Просматривать свои задачи с статусами
- 📅 Смотреть встречи на сегодня или неделю
- ➕ Создавать встречи в календаре Битрикс24

---

## 🚀 Быстрый старт

### 1. Клонируй репозиторий

```bash
git clone https://github.com/твой-юзернейм/bitrix-telegram-bot.git
cd bitrix-telegram-bot
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Создай файл `.env`

Скопируй `.env.example` в `.env` и заполни токены:

```bash
copy .env.example .env
```

Открой `.env` и вставь свои данные:

```
TELEGRAM_TOKEN=токен_от_BotFather
BITRIX_WEBHOOK_URL=https://домен.bitrix24.ru/rest/1/ключ/
```

### 4. Запусти бота

```bash
python bot.py
```

---

## 📋 Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/tasks` | Мои задачи из Битрикс24 с статусами |
| `/calendar` | Встречи на сегодня или неделю |
| `/add_meeting` | Создать встречу (диалог пошагово) |

---

## 🔑 Как получить токены

**Telegram токен:**
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot` и следуй инструкциям
3. Скопируй полученный токен

**Вебхук Битрикс24:**
1. Зайди в Битрикс24 → Приложения → Вебхуки
2. Нажми «Добавить вебхук» → «Входящий вебхук»
3. Включи права: `task` и `calendar`
4. Скопируй URL вебхука
